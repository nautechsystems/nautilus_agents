# nautilus-agents

![license](https://img.shields.io/github/license/nautechsystems/nautilus_agents?color=blue)
[![Discord](https://img.shields.io/badge/Discord-%235865F2.svg?logo=discord&logoColor=white)](https://discord.gg/NautilusTrader)

Agentic infrastructure for [NautilusTrader](https://nautilustrader.io).

This crate provides foundational types and traits for building autonomous
agents on top of the Nautilus trading engine.

**Status**: early development. The API is not yet stable.

## Platform

[NautilusTrader](https://nautilustrader.io) is an open-source, high-performance, production-grade
algorithmic trading platform, providing quantitative traders with the ability to backtest
portfolios of automated trading strategies on historical data with an event-driven engine,
and also deploy those same strategies live, with no code changes.

NautilusTrader's design, architecture, and implementation philosophy prioritizes software
correctness and safety at the highest level, with the aim of supporting mission-critical,
trading system backtesting and live deployment workloads.
